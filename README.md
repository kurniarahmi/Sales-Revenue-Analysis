# Sales-Revenue-Analysis
Project ini bertujuan untuk menganalisis performa penjualan dan revenue menggunakan dataset e-commerce publik. Analisis difokuskan untuk mengidentifikasi tren pertumbuhan serta faktor utama yang mempengaruhi revenue dari sisi produk, customer, dan region.

Pendekatan yang digunakan mencakup eksplorasi data menggunakan SQL serta analisis lanjutan menggunakan Python untuk menghasilkan insight yang lebih mendalam dan actionable.
## Objectives
* Mengidentifikasi tren revenue dari waktu ke waktu
* Menemukan faktor utama yang mempengaruhi kenaikan/penurunan revenue
* Menganalisis performa berdasarkan:
  * Produk
  * Customer
  * Region
* Memberikan insight dan rekomendasi berbasis data

## Tools & Technologies
* SQL (BigQuery) → Data exploration & querying
* Python (Pandas, Matplotlib/Seaborn) → Deep analysis & visualization
* Dataset: TheLook E-commerce (BigQuery Public Dataset)

## Analysis Breakdown
1. Revenue Trend Analysis
   Analisis tren revenue bulanan untuk melihat pola pertumbuhan dan momentum bisnis.
2. Product Performance
   Identifikasi produk dengan kontribusi revenue tertinggi dan terendah, serta analisis distribusi performa produk.
3. Customer Segmentation
   Segmentasi customer berdasarkan total spending untuk memahami distribusi nilai pelanggan.
4. Regional Analysis
   Evaluasi kontribusi revenue dari berbagai negara untuk mengidentifikasi market utama.
5. Retention Analysis
   Analisis perbandingan antara repeat dan one-time customers untuk mengukur tingkat retensi.

> Project ini bersifat exploratory dan bertujuan untuk menunjukkan kemampuan dalam data analysis menggunakan SQL & Python, business thinking & insight generation, dan data storytelling untuk pengambilan keputusan
